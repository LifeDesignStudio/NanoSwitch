# NanoSwitch - Fast, minimal switching for macOS

## See every window. Switch in one move.

macOS's built-in Cmd+Tab cycles through apps.

NanoSwitch shows a thumbnail grid of every **window** — so you jump directly to the exact one you need.

---

## How it works

Hold `Cmd+Tab` and a thumbnail grid appears.

Use Tab or arrow keys to navigate, then release Cmd to switch.

No new shortcuts. No learning curve.

---

## Features

- **Window-level switching**
  See all open windows individually — including multiple windows from the same app.

- **Thumbnail previews**
  See the content of each window before you switch to it.

- **App icons on thumbnails**
  Each thumbnail shows the app icon for quick identification.

- **Close windows directly**
  Hover any thumbnail and click ×, or press `W` to close a window without leaving the switcher.

- **Current Space only**
  Only windows from your current desktop are shown. No clutter.

- **Lightweight**
  Runs quietly in your menu bar with minimal CPU and memory usage.

## Permissions

NanoSwitch requests two permissions on first launch.

| Permission | Purpose |
|---|---|
| Accessibility | Intercept Cmd+Tab system-wide |
| Screen Recording | Capture window thumbnails |

The app works without Screen Recording — windows will show app icons instead of thumbnails.

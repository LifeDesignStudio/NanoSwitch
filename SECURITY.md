# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 1.x     | ✓         |

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not** open a public GitHub issue.

Instead, report it via one of the following:
- **GitHub**: Open a [private security advisory](https://github.com/LifeDesignStudio/NanoSwitch/security/advisories/new)

We will respond within 7 days and aim to release a fix within 30 days of confirmation.
